# Fallout 2 tools

## TL;TR

```
make
ln -s /path/to/fallout2/MASTER.DAT MASTER.DAT
```

- `bin/critical`
  - edit critical hit chance and critical hit bonus

- `bin/dat2dump`
  - dump objects from `*.DAT`

- `bin/inventory`
  - edit inventory

- `bin/perks`
  - edit perks

- `bin/save_info`
  - get info from a `SAVE.DAT`

- `bin/special`
  - edit SPECIAL

- `bin/traits`
  - edit traits
